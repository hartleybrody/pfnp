Starting to use a grid framework

One of the most important things we can use CSS for is laying out the structure of our site. When building a new website from scratch today, a lot of developers use a CSS grid framework that someone else wrote and released for free. This saves us from having to write a bunch of "boring" code that handles the grid and columns, and making sure things look good on mobile. That way we can focus on writing the HTML and CSS that's specific to our website.

This page uses Twitter's bootstrap CSS framework. More information about that can be found here. The basic premise is that you have rows of content that should be stacked horizontally. Then within a row, you have columns of various widths (this text and the sidebar next to it, for example). Each row can be split up into up to 12 columns. This text is in a column that is 8 units wide, and then the sidebar is in a column that is 4 units wide.

Your goals

  1. Add another row with several different sized columns.
  2. Adjust the widths of the columns in this row, so that the main text and sidebar are the same width.
  3. Make the columns change size on different screen widths. You'll need to read the bootstrap documentation for this.
